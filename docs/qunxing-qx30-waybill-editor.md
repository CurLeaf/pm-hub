# 群兴 QX-30：面单编辑器接入（完成说明）

effort：**一般** · 负责人：@嘉松 · 完成：2026-04-28

## 目标

在群兴业务侧接入**面单 / 物流标签模板编辑器**，使运营可配置打印样式，并与订单发货、面单打印流程衔接。

## 实现摘要

1. **编辑器接入**  
   在管理端嵌入或跳转至面单编辑能力（与现有打印/物流模块约定一致），支持模板的新增、编辑、预览与保存。

2. **数据与配置**  
   模板定义（字段、布局、纸张规格等）按后端约定持久化；列表与当前店铺/物流商维度绑定关系以实现为准。

3. **使用入口**  
   在「已发货 / 待发货 / 面单相关」等业务菜单中提供可达入口，避免深链与权限遗漏。

4. **与打印联动**  
   打印面单时读取当前生效模板；若无模板或接口异常时有明确提示（与 QX-35「已发货订单：批量打印修复」等后续项可区分边界）。

## 验收（回顾）

- 运营可完成：**创建模板 → 保存 → 在打印流程中选模板 → 输出与预览一致**（以线上环境为准）。  
- 权限与数据范围符合分中心 / 店铺隔离规则（与产品既有设计一致）。
